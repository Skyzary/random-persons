A simple web project that generates and displays random user cards using the [Random User API](https://randomuser.me/).

This project was created to practice asynchronous JavaScript (Async/Await), DOM manipulation, and the Vite bundler.

## üì¶ Installation and Setup

To run the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vladimirlevkovskyi/random-persons.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd random-persons
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start development mode:**
   ```bash
   npm run dev
   ```
   Open the link that appears in the terminal (usually http://localhost:5173).
